## Python环境安装 
1. 今天安装python2.7/3 环境. 官网下载地址:  https://www.python.org/downloads/release/python-2710/
2. 注意如下问题
    1. win8.1 使用管理员身份运行cmd,否则就算系统TEMP变量增加路径后仍然出现找不到pthon的提示.
    2. 程式代码要求对空格的要求之严格. 不能随意缩进. 譬如raw_input之后后续不该缩进但是缩进了, 会提示unexpected indent.
    indent 缩进,排班. 切割...使锯齿状的意思. 还有订单,订购的意思.
    3. 关于注释: 三个引号,多行注释. #号单行注释.
3. 系统环境变量的配置.
    修改path路径增加只想Python36文件夹的路径， 注意以`/`结尾。

## Python运行环境的配置
> 更新2016/2/17 如下设置可以放在编译的文件夹内完成，默认不添加的情况下，配置系统path之后，就可以编译了， 添加文件反而多了一个编译。
所以如下步骤可忽略， 直接配置好path， 然后**重启**sublime即可。

Sublime `Perferrences->Browse Packages`打开文件夹目录，新建一个文件名为`Python.sublime-build`文件，内容为：
    ```
{ "cmd":["python.exe", "-u", "$file"], "path":"C:/Python27", "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)", "selector": "source.python"}
    ``` 
至此， 可以使用·`Ctrl + B`完成Python文件的编译及运行。 注意：交互类型的（比如需要输入）运行就会报错。
