单纯赋值引用
浅拷贝
深拷贝
a 原始列表, a[3]为列表, 其他原则为值类型
b 赋值引用, 同一个东西
c 浅拷贝
d深拷贝

  1. >>> a=[1,2,3,[4,5]]
  2. >>> b= a
  3. >>> import copy
  4. >>> c=copy.copy(a)
  5. >>> d=copy.deepcopy(a)
  6. >>> b
  7. [1, 2, 3, [4, 5]]
  8. >>> c
  9. [1, 2, 3, [4, 5]]
  10. >>> d
  11. [1, 2, 3, [4, 5]]
  12. >>> a.append(6) --增加值元素后, 引用变更,深浅拷贝均不改变,因为父类已copy.
  13. >>> b
  14. [1, 2, 3, [4, 5], 6]
  15. >>> c
  16. [1, 2, 3, [4, 5]]
  17. >>> d
  18. [1, 2, 3, [4, 5]]
  19. >>> a[0]=2--针对值原则变更, 引用变更, 深浅拷贝均不可改变, 因为只copy父类.
  20. >>> a
  21. [2, 2, 3, [4, 5], 6]
  22. >>> b
  23. [2, 2, 3, [4, 5], 6]
  24. >>> c
  25. [1, 2, 3, [4, 5]]
  26. >>> d
  27. [1, 2, 3, [4, 5]]
  28. >>> a[3] 
  29. [4, 5]
  30. >>> a[3].append(7)--针对引用类型变更, 引用变更, 浅拷贝均变更, 因为只复制父类.
  31. 对于深拷贝, 还是不变更.
  32. >>> a
  33. [2, 2, 3, [4, 5, 7], 6]
  34. >>> b
  35. [2, 2, 3, [4, 5, 7], 6]
  36. >>> c
  37. [1, 2, 3, [4, 5, 7]]
  38. >>> d
  39. [1, 2, 3, [4, 5]]
