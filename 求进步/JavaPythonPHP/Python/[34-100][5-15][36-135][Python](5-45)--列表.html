列表是引用类型, 赋值以后, 一变都变, 只是引用类型.
例如:
list和listmile 赋值引用后, 要变都变的特性!
  1. >>> list
  2. ['milo', 30, 'male']
  3. >>> listmile
  4. ['milo', 30, 'male']
  5. >>> list.remove(listmile[1])
  6. >>> list
  7. ['milo', 'male']
  8. >>> listmile
  9. ['milo', 'male']
  10. >>> id(list)
  11. 45091592L
  12. >>> del(list[1])
  13. >>> list
  14. ['milo']
  15. >>> listmile
  16. ['milo']

元组是值类型,无法更改部分, 全部更改,可以,知识重新地址空间赋值而已. 可以理解为,用完即废.
