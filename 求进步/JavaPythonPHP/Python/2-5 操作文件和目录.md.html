## 基本模块功能
```
    import os 
    os.name #'nt'表示windows操作系统
```
其他windows不可用的距离, 说明os模块和操作系统是相关联的.
```
    os.uname()
    os.environ()
    os.getenv()
```
## 操作文件和目录
```
    os.path.abspath('.') #查看当前目录的绝对路径
    os.path.join('/users/administrator','testdir') #创建目录步骤1 新目录完整路径表示出来
    os.mkdir('users/administrator/testdir') #创建一个目录
    os.rmdir('users/administrator/testdir') #删除一个目录

    #目录的合并,拆分都是根据字符串操作, 不要求一定存在
    os.path.split('/user/administrator/testdir/file.txt') #拆分结果:('user/administrator/testdir','file.txt')
    os.path.splittext('user/administrator/file.txt') #结果('user/administrator/file'.'txt') 
    os.rename('file.txt','test.py') #对文件重命名
    os.remove('test.py') #删除文件 
```
利用python的特性来过滤文件.
```
    [x for x in os.listdir('.') if os.path.isdir(x)] #当前目录下的所有文件夹/目录
    [x for x in os.listdir('.') if os.path.isfile(x) and os.path.splitext(x)[1]=='.py'] #列出所有的.py文件.
```
