全局变量, 函数作用于覆盖.
可以在函数体里面声明: global a, 成为全局变量;
函数里重名global后, 全局变量重新赋值. 注意, 在函数执行前, 仍然为原值!!

函数调用:
  1. >>> def f(x,y):
  2. 	print "%s : %s" %(x,y)
  4. >>> f(*t)
  5. name : milo
  6. >>> t
  7. ('name', 'milo')
其中, (x,y)需要以元组形式.
