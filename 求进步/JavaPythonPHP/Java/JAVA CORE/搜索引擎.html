其实我们日常使用的搜索引擎例如百度，Google，我们一般情况都是直接搜索。其实这些搜索引擎有一些鲜为人知的高级搜索语句，适当使用这些语句会带给我们很大的方便！好，我们现在就开始认识这些语句吧！熟悉它们后，你可以直捣网站的资源宝地下载，可以只看指定站点的文章，可以让搜索结果匹配无比，还可以让你只看到你想看的结果！

3.“”双引号让百度搜索结果精确匹配

　　如果输入的查询关键字很长时候，百度在经过分析后，给出的搜索结果中，可能是拆分的。如果你对这种情况不满意，可以尝试给关键字加上双引号，那么百度就会返回最精确匹配的结果给你了。

　　例如，搜索 IT世界网学院频道 ，如果不加双引号，搜索结果会被拆分，但加上双引号后“IT世界网学院频道”，那么获得的结果就全是符合要求的了。

4.-减号让你只看只想看的东西

　　如果你发现搜索结果中，有某一类网页是你不希望看见的，而且，这些网页都包含特定的关键词，那么用减号语法，就可以去除所有这些含有特定关键词的网页。

　　例如，搜索越狱，希望是关于越狱介绍或音乐等信息的话，却发现开头大部分方面的网页是关于越狱视频的。那么我们就可以在搜索时，输入 越狱 -视频，这时搜索结果将减去关键字为视频关于越狱的文章了！

　　注意：前一个关键字与减号之间需要空格，不然就会把减号当成是连字符处理！
