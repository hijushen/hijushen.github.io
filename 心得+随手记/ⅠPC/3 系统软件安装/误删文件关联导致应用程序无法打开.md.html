## 故事背景
2016年农历新年过后开工第一天，由于右键一直显示忙等待的频繁闪烁图片， 网络查询问题出现在> 删除文件残留了文件右键菜单，导致了这种情况 > 。开开心心的使用的软媒的右键菜单管理删除了某种。 结果导致所有的.exe都无法运行，cmd都不能运行，这下麻烦大了，不能运行程序那不是什么都做不了么。 
## 解决方案
新建文本文档，重命名为.reg结尾，导入手写的注册表文件，恢复注册表键值。
```
Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.exe]
@="exefile""Content Type"="%1, %x"
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.exe\PersistentHandler]
@="{098f2470-bae0-11cd-b579-08002b30bfeb}"
[HKEY_CLASSES_ROOT\exefile\shell\open\command]
@="\"%1\" %*"
```
注意，第一行也不能省略，否则无法运行，重要的是第三个键值。
