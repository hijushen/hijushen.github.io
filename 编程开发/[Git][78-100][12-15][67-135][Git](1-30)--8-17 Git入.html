2. Git 分布式仓储的优点. 离线方式 去中心化, 大家都能拥有所有仓库... (权限, 后续了解).
3. Windows 安装: 从http://msysgit.github.io/ 网址下载, 默认配置安装. 
4. 新建一个空目录, 或者在已有目录下, 电子Git init here 也可以. 掩饰在D:\git 下打开git bash.
5. 输入命令: git config --global user.name "liyu";
                    git config --global user.email "hijushen@qq.com".
    全局配置, 也可以针对仓库进行配置. 此处不详细介绍. --global 代表全局.
6. 创建版本库, 又撑仓库: repository. pwd 用于显示当前目录, 在bash下.
7. git init建立空的仓库. 目录下多出隐藏目录.git 文件夹.
8. ls -ah 显示当前目录文件和文件夹. 分别为 . .. .git test.txt.
9. 1) git add 文件名.txt  2) git commit -m "说明内容". 可以多次add , 一次提交.
10. git status 和git diff 文件名.txt 用于查看当前状态和比较究竟修改了什么. 
11. git log 查看文件历史版本功能. 可以使用git reset --hard HEAD^ 返回到上一版本. HEAD^^代表上上一版本. HEAD~100代表                       上100版本.  也可以根据 commitID 进行回退.  git  reflog查看命令历史, 回退到任意版本.
12.  git log 可选参数--pretty=oneline. oneline用于显示单行, 回退是可以输入前几个数字.
13. git reflog 查看所有历史版本操作记录.
14. 使用git reset --hard head^ (head~100) 具体版本号就都可以回复到之前状态了.
