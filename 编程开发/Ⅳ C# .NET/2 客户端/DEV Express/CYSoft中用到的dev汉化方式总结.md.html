## 入口
在program.cs或者启动之前, 设置:
```
  1. CYSoft.Platform.Controls.Localizers.Localizer.InitLocalizer(); //静态方法设置.
```
## 具体设置
对应代码 , 主要的是设置对应空间`Localization`的`Active`静态属性为 自定义编写的类.
```
  1. DevExpress.Accessibility.AccLocalizer.Active = new CYSoft.Platform.Controls.Localizers.AccLocalizer_zhchs();
  2. DevExpress.XtraCharts.Localization.ChartLocalizer.Active = new CYSoft.Platform.Controls.Localizers.ChartLocalizer_zhchs();
  3. DevExpress.XtraGrid.Localization.GridLocalizer.Active = new CYSoft.Platform.Controls.Localizers.GridLocalizer_zhchs();
  4. DevExpress.XtraLayout.Localization.LayoutLocalizer.Active = new CYSoft.Platform.Controls.Localizers.LayoutLocalizer_zhchs();
  5. DevExpress.XtraBars.Localization.BarLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraBarsLocalizer_zhchs();
  6. DevExpress.XtraEditors.Controls.Localizer.Active = new CYSoft.Platform.Controls.Localizers.XtraEditorsLocalizer_zhchs();
  7. DevExpress.XtraNavBar.NavBarLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraNavBarLocalizer_zhchs();
  8. DevExpress.XtraPrinting.Localization.PreviewLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraPrintingLocalizer_zhchs();
  9. DevExpress.XtraReports.Localization.ReportLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraReportsLocalizer_zhchs();
  10. DevExpress.XtraRichEdit.Localization.XtraRichEditLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraRichEditLocalizer_zhchs();
  11. DevExpress.XtraTreeList.Localization.TreeListLocalizer.Active = new CYSoft.Platform.Controls.Localizers.XtraTreeListLocalizer_zhchs();
```
