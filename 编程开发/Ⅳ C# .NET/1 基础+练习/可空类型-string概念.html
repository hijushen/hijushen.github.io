String str1 = null; string str2 = ""; string str3 = str2; 的内存分配情况。

"" 和 string.Empty基本相同， 后者是语法的优化形式。
在堆上和栈上都会分配空间， 堆上空间长度为0.

而null 是仅仅在栈上声明空间， 不指向任何地方。堆上不分配。 

string str3 = str2; 会优化， 指向同样一段长度为0 的空间.
