# 极客学院Linq教程
## 使用Linq的2种方式
- Query syntax 语句表达式
- Method syntax 拓展方法 +　Lambda 表达式。
- 代码如下:
    ```
            int[] numbers = { 5, 10, 8, 3, 6, 12 };

            //1.Query syntax
            var numQuery1 = from num in numbers
                            where num % 2 == 0
                            orderby num
                            select num;

            //2.Method syntax
            var numQuery2 = numbers.Where(n => n % 2 == 0).OrderBy(n => n);



            Array.ForEach(numQuery2.ToArray(), p => { Console.Write(p.ToString()+" "); });
            Array.ForEach(numQuery1.ToArray(), p => { Console.Write(p.ToString() + " "); });


            Console.ReadLine();  
    ```

## Linq的基本组成
-`ToList()``Count()``ToArray()`都会导致执行结果执行

## Linq的分组等操作.
- `group .. by``join .. on`的使用
```
List<Customer> customers = new List<Customer>();
            customers.Add(new Customer() { Name = "Jack", City = "Beijing" });
            customers.Add(new Customer() { Name = "LiLie", City = "Beijing" });
            customers.Add(new Customer() { Name = "HanMeimei", City = "Shanghai" });

            List<Employee> employees = new List<Employee>();
            employees.Add(new Employee() { Name = "Jack", ID = 101 });
            employees.Add(new Employee() { Name = "Emily", ID = 102 });

            var queryCustomers = from c in customers
                                 group c by c.City;

            Array.ForEach(queryCustomers.ToArray(), c =>
            {
                Console.WriteLine(c.Key);
                Array.ForEach(c.ToArray(), cg =>
                {
                    Console.WriteLine(" {0}",  cg.Name);
                });
            });


            var queryJoin = from c in customers
                            join e in employees on c.Name equals e.Name
                            select new { PersonName = c.Name, PersonID = e.ID, PersonCity = c.City };

            Array.ForEach(queryJoin.ToArray(), p =>
            {
                Console.WriteLine("{0},{1},{2}", p.PersonName, p.PersonCity, p.PersonID);
            });

            Console.ReadKey();
        }

        class Customer
        {
            public string Name { get; set; }
            public string City { get; set; }
        }

        class Employee
        {
            public string Name { get; set; }
            public int ID { get; set; }
        }
```

## 预处理`into`和`let`的使用
- `into`结果成中间变量
- `let` 也是预处理, 处理后保存为中间变量

```
        var queryCustomers = from c in customers
    group c by c.City into cusGroup//中间变量cusGroup依然可以筛选
    where cusGroup.Count() >= 1    
    Array.ForEach(queryCustomers.ToArray(),P =>{
        Console.WriteLine("{0},{1}",P.City,P.Count);
    });
    
    var stringQuery = from s in strings
                      let words = s.Split(' ')//1.预处理成单词模式
                      from word in words
                      let w = word.ToUpper()//2.预处理每个单词成大写
                      select w;
    foreach (var s in stringQuery)
    {
        Console.WriteLine(s.ToString());
    }
```
