## 标准查询运算符
### LINQ如何实现,构造一个LINQ查询
- from在前select在后,二者之间可以找到join, where, orderby 以及into.
 
### 筛选信息
- where字句的传统筛选器,代码:
```
var numbers = new int[] { -1, -32, 3, 5, -8, 13, 7, -41 };
var magnitude = from n in numbers where Math.Abs(n) > 5 select n;
foreach (var m in magnitude)
{
    Console.WriteLine(m);//-32,-8,13,7,-41
}
```
- OfType用于筛选出能够转换成特定类型的值.
```
var randomData = new object[] { 1, "two", 3, "four", 5.5, "six", 7M };
var canBeInt = from r in randomData.OfType<int>() select r;
foreach (var i in canBeInt)
{
    Console.WriteLine(i);//1,3. 转换一个序列中的类型,与Convert类不同.7M代表十进制7,但是OfType并不会自动转换.
}  
```
 
### 使用限定符
- 限定符All,Any以及Contains都是拓展方法,他们均返回一个布尔值,指示集合中部分或者全部元素是否满足限定符的参数条件.
 
### 利用Skip和Take实现分区操作
- Skip,SkipWhile,Take以及TakeWhile用于将集合拆分成两部分并返回其中的一个.
- 1-10的序列,`Take(5)`表示返回前5个元素的序列, `Skip(3)`表示返回含有后7个元素的序列. `TakeWhile`和`SkipWile`使用的是测试条件而不是基数值.
 
### 使用生成运算
- 包括`DefaultIfEmpyt`,`Empty`,`Range`以及`Repeat`. 生成运算用于创建新的值序列.
- `DefaultIfEmpyt``Empty`.代码:
```
var empties = Enumerable.DefaultIfEmpty<char>(Enumerable.Empty<char>());
Console.WriteLine("Count:{0}",empties.Count());
Console.WriteLine(empties.ElementAtOrDefault<char>(0));//.ElementAt<char>(0)); //都是空
```
 
- `Range`和`Repeat`,随机产生及重复生成.代码:
```
public class Customer
{
    public int ID { get; set; }
    public string CompanyName { get; set; }
    public string City { get; set; }
    public int RanID { get; set; }
}
```
```
Action<Customer> print = c =>
Console.WriteLine("RanID:{3}\nID:  {0}\n Customer: {1}\nCity: {2}\n",
c.ID,c.CompanyName,c.City,c.RanID);
var customers = Enumerable.Repeat<Customer>(new Customer{
RanID = new Random().Next(1,100),
ID = DateTime.Now.Millisecond,
CompanyName = "Test Company",
City = "Test City"},100); //浅拷贝又一证明, ID和RanID都是一样的, 并没有产生值不一样.100个引用而已.
Array.ForEach(customers.ToArray(), c => print(c));//完全一样的值.
```

### 从序列中获取特定元素
- 元素方法包括`ElementAt`,`ElementAtOrDefault`,`First`,`FistOrDefault`,`Last`,`LastOrDefault`,`Single`以及`SingleDefault`. 他们均从序列中返回单个元素.
