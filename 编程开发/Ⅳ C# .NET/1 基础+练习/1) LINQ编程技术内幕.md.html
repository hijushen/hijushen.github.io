## 匿名类型
- 简单匿名类型
`var title = "hello world"`
- 数组初始化器 `var fibonacci = new int[] {1,1,2,3,5,8,13,21}`.
- 匿名类型: 
    - `var dena = new {First= "Dena", Last = "Swanson"};`
    - 匿名类型添加方法: `Fun<string,string,string> Concat1 = delegate(string first,string last){return last + ", " + first;};`
    - for语句使用匿名类型索引, 举例说明一个`foreach`类型迭代语句:`foreach(var fibo in from f in fibonacci where f%3==0 select f` 其中 `from f in fibonacci where f %3 ==0 select f`返回了一个`IEnumerable<T>`类型的枚举.
    - 从函数返回匿名类型, 不推荐使用.
- 匿名方法

## 复合类型初始化
- 通过命名类型初始化对象. 
    - 初始化匿名类型并重命名:
    ```  
    var names = from p in people select new {FirstName = p.Name};
    foreach(var name in names)
        Console.WriteLine(name.FirstName);
    Console.ReadLine();
    ```

- 使用转换运算符
    - ToArray. `Array`很快但是,编写数组管理代码很麻烦, 可以使用类似`List<T>`管理数据, 然后将需要的数据返回. 代码类似: `var points = from p in this select p.MyPoint; return points.ToArray<PointF>();`
    - `OfType<T>`返回一个`IEnumerable<T>`类型集合. 代码示例: `var numbers = new object[]{1,"two",null,"3","4"}; foreach(var arInt in numbers.OfType<int>()){Console.WriteLine(areInt);};` 返回1和4,只是Int类型的类型.
    - `Cast` 返回异常:`foreach(var anInt in numbers.Case<int>())`, 当Cast运算符遇到字符串"two"时将产生一个InvalidCastException异常.
    - `AsEnumerable` 先使用`AsEnumerable`方法再调用`Where`的话, 调用的就是`IEnumerable`的`Where`方法而不是后来重新实现.
    - `ToList` 强制进行计算, 并返回一个`List<T>`类型
    - `ToDictionary` 代码`IDictionary<string,Games> stats = spartans.ToDictionary(key=>Key.Opponent);`
    - `ToLookup`. 用于将`IEnumerable<T>`转换成`Lookup<Key, Element>`类型, Lookup类似于Dictionary. Lookup将键与一组值映射起来, Lookup没有公共构造函数, 而且不可变. 创建Lookup之后, 不嗯给你增加或者删除其中元素或键.示例代码:
    ```
    public class Product
        {
            public string Code { get; set; }
            public string Description { get; set; }
        }

        static void Main(string[] args)
        {
            List<Product> products = new List<Product>
            {
                new Product{Code="SNM1",Description = "Spam"},
                new Product{Code="SNM2",Description = "dfdfs"},
                new Product{Code="LME1",Description="Bologna"},
                new Product{Code="SUP1",Description="Tomato"},
                new Product{Code="SUP2",Description="Chicken Moodle"}
            };

            Lookup<string,string> lookup = (Lookup<string, string>)
                products.ToLookup(c => c.Code.Substring(0, 3), c => c.Code + " " + c.Description);

            foreach (IGrouping<string,string> group in lookup)
            {
                Console.WriteLine(group.Key);
                foreach (string s in group)
                {
                    Console.WriteLine("  {0}",s);
                }
            }
            
            Console.ReadLine();

            IEnumerable<string> spmGroup = lookup["SNM"];
            foreach (var str in spmGroup)
            {
                Console.WriteLine(str);
            }

            Console.ReadLine();
        }
    ```
