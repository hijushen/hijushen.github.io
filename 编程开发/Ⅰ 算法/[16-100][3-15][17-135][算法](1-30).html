快速排序: 
时间平均复杂度. nlogn. 最坏的情况, 演变成冒泡排序. 复杂度n2.
空间复杂度1.

关键字: 挖坑填数 + 分而治之.
2) http://developer.51cto.com/art/201403/430986.htm (坐在马桶上看算法：快速排序) 
总结句: 快速排序的每一轮处理其实就是将这一轮的基准数归位，直到所有的数都归位为止，排序就结束了。

如下为详细内容,参考"挖坑"代码:
  1.         public int[] QuickSort(int[] array,int len)
  2.         {
  3.             Sort(ref array, 0, len - 1);
  4.             return array;
  5.         }
  7.         public void Sort(ref int[] array, int low, int high)
  8.         {
  9.             int i, j;
  10.             int index;
  11.             if (low >= high)//已相遇,不排序.
  12.                 return;
  13.             i = low;
  14.             j = high;
  15.             index = array[i]; //1-基准坑.
  16.             while(i < j) //不相遇不结束.
  17.             {
  18.                 while (i < j && array[j] >= index)
  19.                     j--;
  20.                 if (i < j)
  21.                 {
  22.                     array[i++] = array[j]; //2-右边的坑填入基准位,右边坑空出. 基准位左移动.
  23.                 }
  24.                 while (i < j && array[i] < index)
  25.                     i++;
  26.                 if (i < j)
  27.                 {
  28.                     array[j--] = array[i]; //3-左边的坑挖出,填右边坑. 左边坑空出.
  29.                 }
  30.             }
  31.             array[i] = index;//4-基准填入坑.
  32.             Sort(ref array, low, i - 1);
  33.             Sort(ref array, i + 1, high);
  34.         }
  35.     }
