GET 和POST 在HTTP中只能是大写；
GET是直接获得地址，获得数据；
POST用于发送 数据，例如表单，让服务器接受附在请求之后的数据；
HTTP响应：在接受和解释请求消息后，服务器会返回一个HTTP响应消息；
HTTP也有3部分组成：状态行、消息报头、响应正文。
          状态行除了结尾的CRLF(回车换行)序列外，不允许出现CR或LF字符；
      所有样式如下4规则：1 浏览器默认，2 External style sheet 外部样式表，
3 Internal style sheet 内嵌样式表，(在<head>标签内)
4 Inline style （inside an HTML element）行内样式（在一html元素内）

往下，优先级越高第四种具有优先性。

The CSS syntax is made up of three parts:a selector,a property and a value:
--css 语法三要素： 选择器，属性，值。 例如： selector{property:value}
选择器是你希望去定义的HTML元素/标签，多个值可以用双引号来扩起来；指定多个属性，使用分号； 来分割。

选择器可以组合，用，好相互间隔；
选择器类可以将同一类型HTML元素定义成不同的样式，必须在HTML文档中使用类属性
