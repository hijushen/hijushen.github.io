## HTML基本标签
- html 开始和技术
- head标签定义特殊内容
    - `<title>`
    - `<meta>` 基本信息,供搜索引擎搜索
        - `<meta name="" content />` name 包括 keywords, 网页关键字,用英文隔开的多个关键字; description 网页描述; author 网页作者; copyright 版权信息
        - `<meta http-eqiv="" content="" />` 内容介绍了2种
            - content-type  `text/html;charset=2312`
            - refresh   `4(秒数);url="http://..."`
    - `<style>` 定义CSS样式
    - `<link>` 外部CSS文件或搅拌
    - `<script>`基本语言>
    - `<base>` 所有链接的基础定位.
 
- body 标签. 可见内容
- HTML 注释: `<!-- -->`
 
## 段落与文字
- 段落与文字
    - `<h1> ~ <h6>`
    - `<p>` 段落标签
    - 换行标签 `<br />`
    - 文本格式化标签.
        - `<b></b>`加粗;
        - `<strong></strong>`;
        - `<i></i><cite></cite><em>`斜体标签, 尽量使用`<em>`;
        - 上标下标`<sup></sup><sub></sub>`;
        - 大字号和小字号标签: `<big></big><small></small>` , 网络化语义中 `<small>`用于网页底部的版权声明.
        - 删除线:`<s></s>`, CSS种使用 `text-decoration:linethrough`实现;
        - 下划线`<u></u>` 实际使用CSS中的`text-decoration:underline` 实现.
        - 水平线标签: `<hr />`
        - `<div></div>` 区域标签
    - 行内标签: `<strong><em><s><u><a><span><img><input>`　代表加粗, 斜体, 删除线, 下划线, 超链接, 行级区域, 图片, 表单等属性.
 
## 列表
- 有序列表: `<ol><li..></ol>`
- 无序列表: `<ul><li..></ul>`
- 定义列表: `<dl><dt></dt><dd><dd></dl>`
 
## 表格
- `<table></table><tr></tr><td></td>` 表格, 单元行, 单元格;
- 表格结构标签: `thead tbody tfoot th` 表头 表身 表脚 表头单元格
- 合并行: rowspan; 和并列: colspan
 
## 图像
- img alt title 属性, alt给搜索引擎看, title给用户看.
- 24位图足够与32位很难肉眼区分; 位图格式: jpg 处理一般图拍了; png支持透明信息, logo或者点缀. 网页专用格式. gif逐帧动画.
- 矢量图格式. .ai .cdf .fh .swf
 
## 超链接
- `<a href="链接地址" target="目标窗口的打开方式"`. target 分: _self 默认方式,当前窗口打开; _blank 全新空白窗口打开链接; _top,_parent 顶层框架和当前框架上一层方式打开.
- 超链接的分类: 外部链接,内部链接;  锚点链接: `<a href="#music">推荐音乐</a><br />` 表示同个页面中只想id为music的标签.
 
## 表单
### `form` 标签
- name属性,
- action 提交到地址属性,
- method 包括 get和post方式;
- target目标显示方式, 包括常用的_self(默认),_blank 新窗口, 和不常用的 _parent,_top分别为父级窗口和当前窗口的所有界面.
- entype 编码属性: `application/x-www-form-urlencoded`默认的编码方式; `multipart/form-data` MIMI 编码, 对于上传文件这种表单,必须选择这种.
 
### 表单对象
- 放入到form标签内的各种标签. 包括input,textarea,select和option.
- input 标签:
    - 单行文本: `type="text"` ; value 默认值,文字; size 长度,字符单位, maxlength最大长度,最大字符数.
    - 密码文本框: `type='passowrd'` 默认同上.
    - 单选按钮:`type='radio'`. 同name时只可以选择一个. value为提交后的值.
    - 复选框: `id = "checkbox1" type='checkbox checked="checked" /><label for="checkbox1"></label>` 需要文字表述, 必须使用<label for..>
    - 表单按钮, 确认,提交,重置: `type="button" value"确认" /><type ="reset value="重置"><type="submit" value="提交"` . 普通按钮配合JavaScript使用, 提交将当前form 内内容提交到服务器处理, 重置按钮清楚form内内容.
    - 图片. `type="image" src=""` , 能用css就不用图片.
    - 隐藏域 `<input type="hidden" value= "10"/>`
    - 文件域 `type="file"` 注意form的enctype要更改, 同form中描述为`multipart/form-data` MIMI 编码.
    - 多行文本框. `<textarea rows="行数" cols="列数">多行文本框</textarea>`
    - 下拉列表: `<select><option></option>...</select>` 表现形式. 常见属性:
        - multiple 默认只选择一个, 选择多个设置为multiple
        - size 下拉后可见的数量
        - option value 和 selected 表示 说明值和是否选中.
 
## 多媒体
- `embed src="多媒体地址" width="播放界面宽度" height="播放界面高度" />` HTML新增的标签, 一般情况下为相对地址;  可使用`hidden = "true" autstart="true" loop="ture"` 来设置为背景音乐.
 
## 框架
- HTML5 设计frameset(标签集标签).  iframe标签. 嵌入一个子界面, 定义宽度高度等. `scrolling`属性可以为auto, 默认左对齐, yes 显示滚动条, no不现实滚动条等.
