主要参考W3C教学：
http://www.w3school.com.cn/html/html_basic.asp 
  1. 1. 标题： <h1></h1>  ~~~ <h6></h6>
  2. 2. 段落： <p></p>
  3. 3. 链接： <a href=http://www.baidu.com">baidu link</a>
  4. 4. 图像： <img src="./pic/w3c.jpg" width="104" height="104" />
HTML元素：
1） 完整的HTML标签： http://www.w3school.com.cn/tags/index.asp 
如：
标签描述
<!--...-->定义注释。
<!DOCTYPE> 定义文档类型。
<a>定义锚。
<abbr>定义缩写。
2） HTML的全局属性：http://www.w3school.com.cn/tags/html_ref_standardattributes.asp 
如：
属性值描述
classclassname规定元素的类名（classname）
idid规定元素的唯一 id
stylestyle_definition规定元素的行内样式（inline style）
titletext规定元素的额外信息（可在工具提示中显示）

  1. <h1></h1> //标题
  2. <p></p>//段落   如上都会自动换行。
  3. <hr /> <!--水平线-->
  4. <br /> <!--不产生新段落的情况下换行-->
