2. toggleClass 的使用

可选参数switch.   ----可选。布尔值。规定是否添加(true)或移除(false)类。
  1. <script type="text/javascript"> //第一种
  2. $(document).ready(function(){
  3.   $("button").click(function(){

  6. });
