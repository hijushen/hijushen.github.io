DOM对象：
  1. var div = document.getElementById("d1");
  2. div.innerHTML = "123";
jQuery对象：
  1. var $div = $("#d1");
互相转换：
  1. //把DOM对象变更为 jQuery对象
  2. var $div = $(div);
  3. $div.html("111");
  5. //jQuery对象变更为DOM对象
  6. //包装集 DOM对象包装起来的集合
  7. var div = $div[0];
  8. //var div = $div.get(0);
