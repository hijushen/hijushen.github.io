5. jQuery常见的属性，较少，为了满足链式方法和函数式编程
  1.         $("#link").css(); //设置行内css的首选方式，同style属性方式.
  2.         $("#link").attr("style", "cls"); //cls可以是css名称，更灵活.
  4.         $("#link").attr("href", "http://www.baidu.com"); //没有特定方法的DOM属性访问方式
  5.         $("#link").removeAttr("href");  //同上一条，此处为移除模式
