## 数值类型
变量名区分大小写.
- 字符串 字符结合,引号括起来.规则:**如果单引号括起来,那么中间的单引号需要转义字符\.双引号同样. `小技巧: 如果经常将HTML代码保存为字符串,使用**单引号**能节约时间, 因为就不用对HTML元素属性两边的双引号进行转译了.`\
- 数值. 不用引号括起来的数字值.
- Boolean. 他们的值只能是true或false. 0 表示false, 1表示true. 0作为数字赋值给某一个变量, 并将其与Boolean值进行比较时,那么0将作为false对待.
- *与变量有关的性能问题* 某个数据使用次数大于1, 那么就应该被缓存起来.

    ## HTML5标准中的网络数据存储方式
    - localStorage与sessionStorage
    类似于cookies的客户端保存数据技术,但是隐私追踪,恶意程序,很多用户在浏览器中关闭该选项. 变成语言内置新方式实现.
    - setItem方法,getItem方法, removeItem方法. 
    -区别: 前者除非开发者或者用户删除浏览器数据都不会消失. 而`sessionStorage`在每次浏览器会话结束(浏览器被关闭时),其中数据自动移除.
    - 以JSON方式存放大量数据
    - 编写健壮的浏览器程序, 检测浏览器的兼容性
    ```
    1. if(typeof(localStorage) === "undefined"){
    2.     //localStorage is supproted, maybe use cookies?
    3. } else {
    4.     //localStorage is supported, use it may be well.
    5.     //add your localStorage code in here.
    6. }
    ```
