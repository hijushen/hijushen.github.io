## Basic Formatting
### Indentation Levels 缩进
    if () {
        for () {
        }
    }
 
### Statement Termination 声明终止
换行可以结束,但是还是推荐`;`结束. 极端的例子:
 
    fuction getData() {
        return
        {
            title: "",
            author: ""
        }
    }
 
这种声明等同于直接return `undefined`对象.
