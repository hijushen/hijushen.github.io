## 变量
- 定义变量-集中在JavaScript文件或者函数头部, 这样, 他们立即被**缓存** 起来.
- 保留字及标识符

## 函数
- 普通函数 `function sayHello(){alert('here there');//alert 其实是window的一个方法, 与函数无太大差异}` 
- 匿名函数. `function() {}` 的方式运行.

## 作用域
- `var`关键字都是函数作用域范围内发生作用, 
- 回调函数,返回数据:
    - 返回单一值.
    - 返回数组. `return [string1, string2]`
- 将函数放入对象, 函数则变成**方法**被使用. 性能的影响, 但是代码组织的形式更加方便了.

## 循环
- **最佳实践** 执行循环前,检测变量值以确保其大于0.避免无谓执行.
```
  1. var contacts = {
  2.   "addressBook": [
  3.     {
  4.       "name":'hillisha',
  5.       "email":"hill@163.com",
  6.     },
  7.     {
  8.       "name":"paul",
  9.       "email":'paul@163.com',
  10.     },
  11.     {
  12.       "name":'visha',
  13.       "email":'visha@163.com',      
  14.     }
  15.   ]
  16. };
  18. var object = contacts.addressBook,
  19.     contactsCount = object.length,
  20.     target = document.getElementsByTagName("body")[0],
  21.     i;
  23. if(contactsCount >0){
  24.   for (i = 0;i<contactsCount;i++){
  25.     var item = object[i],
  26.         name = item.name,
  27.         email = item.email;
  28.     
  29.   target.innerHTML += '<p><a href = "mailto: ' + email +'">'+name+'</a></p>';
  30.   }
  31. }
```
## 条件语句
- if语句. 使用`===`而不是`==`避免出现`false`与`0`相等的情况.
- `if (){} else if(){} else {}` 语句
- `switch` `switch(){case "": ...; case "": ... ; default: ...}` `switch` 大量比较效率略高于if else
