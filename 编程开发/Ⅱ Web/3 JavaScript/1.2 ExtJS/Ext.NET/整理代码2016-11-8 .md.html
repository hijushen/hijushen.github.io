## GridPanel
## 方法
1. `grid.getRowSize();`
2. `grid.pageSize = ..`
3. `grid.store.currentPage = 1; `
4. `var newRecord = grid.store.getNewRecord();  grid.store.remove(newRecord);` 获取增加的一行,并删除，用于cancel动作
5. `var items = gird.data.items; Ext.each(items, function(items){})` 完成遍历相关动作，
6. `var s = grid.selModel.selected; s.length; s.items;`A 获取控件的选择模型及长度，内容数据； `var gre = GridRowEditID` B 获取RowEditPlugin控件的对象，进行编辑的方法：`gre.startEidt(grid.selModel.getLastSelected().index, 1)` C 第一个参数编辑哪一行，上次选择的Index，第二次参数是focus哪一列。

## Ext:Store
结构：
### Proxy
- `<ext:AjaxProxy>` Url 定义地址，只写方法为当前页面路由下控制器方法；
    - `<ActionMethods Read = "Post" />` 标识获取数据方式
    - `<Reader><ext:JsonReder ..></ext:JsonReader></Reader>`用于表述读取字符方式， `TotalProperty`和`Root`用于标识数据汇总属性和读取根节点. 如果分页，总数为`total`，JSON对象读取的名称为`Root`.
### Model
- <ext:Model>
    - `<Fields><ext:ModelField><ext:ModelFields></Fields>` 结构定义实体
### Listeners
- `Load`
- `SetDefaultSelectRow()`方法, 数据重新加载后根据hidden的id选中对应行.
