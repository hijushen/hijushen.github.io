### MessageBus 用法
- 消息订阅机制, ExtJS中没有. 非原创, dojo中早实现.
- 通过代码,发布-订阅消息. 
```
Event.net.Bus.publisg("Custom.Event1",{item:this});//发布
Event.net.Bus.subscribe("Custom.*",function(){console.log(arguments);});});//订阅
```
- 通过控件,发布和订阅消息.
```
<ext:Panel ...
    <MessageBusListeners>
        <ext:MessageBusListener
            Name="App.*"
            Handler="this.body.createChild({
            html: 'Received event from ' + data.item.id,
            tag: 'p'
            });" />
    </MessageBusListeners>
    <Buttons>
        <ext:Button ID="Button1" runat="server" Text="Publish event 1">
            <Listeners>
                <Click BroadcastOnBus="App.event1" />
            </Listeners>
        </ext:Button>
    </Buttons>
</ext:Panel>
```
- 服务端的消息发布和订阅处理

### `DirectEvent`功能.
- 服务端事件处理。
- 控件里面简写 `onDirectClick`, 通用方法`<DirectEvents>`标签
- 缓冲层`<EventMask>`标签, 参数`<ExtraParams><ext:Parameter /><./>`等标签; 还有就是服务端数据回复及回调. 使用`ExtraParamsResponse`参数.
- 提交钱<Confirmation>`出发前确认按钮.

### `DirectEvent`高级使用
- 对asp.net控件, HTML元素, 调用WebService, 一般处理程序及 更新ASP.NET控件
- 更新ASP.NET控件需要使用`Update()`方法, 更新才生效.
- 给HTML元素增加 事件 也是同样的过程.
- 生成`Ext.net`控件并重新布局. 
- webService服务和ashx一般处理程序返回方式. 
    - WebService服务方法返回 `return new DirectResponse()`, 类使用注解:`[Syste.Web.Script.Services.ScriptService]`, 此注解表示允许 ASP.NET AJAX 从脚本中调用此Web服务.
    - 一般服务返回`new DirectResponse().Return()`类型.
    - 如上2点, 未彻底理解返回值的意义, 初步理解应该是方法中使用`url:`参数采用的异步方式,其实是Ajax 方法, 区别与传统使用. 与`Ext.Net`交互, 所以需要定义返回值类型.

### `DirectMethod`功能
-  通过DirectMethods , 在Js中通过异步方法调用的方式执行服务端方法.
    -  步骤包括: 后台方法定义, 命名控件设定(主要是注解的使用), 以及方法回调的作用.
    -  同步方法的使用, 使用注解即可. 
    -  提升性能: 使用静态方法, 无法根据ID直接访问Form数据. 不能直接访问Request, Response等数据(可以通过HttpContext.Current, 这个还不知道啥用, 要了解). 好处是Ext.net 不会将页面数据提交给服务器, 大大减少了数据传输和服务器执行的事件. 利用回调处理处理. 访问界面控件的方式`App.myLabel.setText(result)`... 待了解: Ext.net控件的顶层类系统, 组织方式. 至少知道怎么访问及使用的吧 . 
- `Ext.net.DirectMethod.request`调用WebService等方法. 适用于一切WebService, 一般处理程序, MVC等通过url访问的东西, 好用.
- 异常处理: `DirectMethods`方法注解的过程中, 自定义`ResourceManager.AjaxSuccess=false;ResourceManager.AjaxErrorMessage=GetFriendlyMessage(e)`的方式.
