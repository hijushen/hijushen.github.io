## 数据类型
### 数据类型
包括`string number boolean undefined null object` 类型.
- float int 都属于number类型
 
### 对象类型
  对象(属性集合), 数组(有序列表), 函数(可执行代码).
- `typeof(arryay)` -> object ; 奇怪? 对象和数组区别不明显,同一类型,但是行为不同.
- `typeof(function)` -> function类型
 
### 基本类型与对象的转换
- 基本类型 -> 对象object
  - `string`基本类型与`String`对象的关系:
    - `typefof(string)`->string 类型; `typeof(String)` -> object 类型;
    - 对于str.length时会自动包装string类型为String类型,从而获得对象的length属性,最后临时的对象呗释放.
 
- 对象object -> 基本类型.
  - 首先,调用对象的ValueOf()方法去的对象的值,如果上下文类型匹配则使用该值;
  - 其次,如果ValueOf()取不到该值,调用对象的`toString()`方法. 如果上下文需要一个数值型,则将该字符串转换为数值.
  - 总结: 上下文推测方式. `valueOf()`优先度大于`toString()`.  V
  - `valueOf()`的作用将对象转换为符合上下文需求的基本类型. `toString()`打印对象对应的字符串,前提是"重载"了`Object`的`toString()`方法.
  - 使用: 上下文需要`boolean`类型, 则所有非空对象为`true`, 其余为`false`.
