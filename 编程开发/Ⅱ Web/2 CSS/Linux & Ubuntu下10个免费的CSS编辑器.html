    Linux & Ubuntu下10个免费的CSS编辑器

    2016-04-05 前端开发 前端开发

    分享Web前端开发中HTML5、JavaScript、CSS3编程学习知识

    来自：慧都控件网

    链接：https://www.evget.com/article/2015/11/9/22928.html

    原文：http://www.smashingapps.com/2015/06/15/10-free-css-editors-for-linux-ubuntu.html


    你正在使用Linux和Ubuntu，却不知道在哪里找到一些用于Linux和Ubuntu的优秀并且免费的CSS编辑器吗？如果是这样，那么小编今天就为大家解决这个问题啦！我们为大家展示了在Linux和Ubuntu下top 10的CSS编辑器！在这个汇总的列表中，我们展示了最好的一些CSS代码编辑器。还在等什么，赶紧一起看起来吧！


    Bluefish


    Bluefish是进行Web开发时最受欢迎的IDE之一，它能够处理编程和标记语言，但是该工具的重点用途在于创建动态和交互式网站。和许多Linux应用程序一样，它是一个轻量级工具，运行速度快，可以一次打开多个文档。同时Bluefish还非常重视UI，任何初学者都能够很快上手，并在不断积累中发现和掌握它的其他功能。




    jEdit


    jEdit是一个在GPL下发布的文本编辑器，它可以在Windows、Linux、Mac OS等多种平台下运行，并且有很多插件，可以扩充基本功能。同时jEdit还支持80多种文件类型的文法加亮显示，支持包括UTF-8在内的多种字符编码。




    TinyMCE


    TinyMCE是一个轻量级的基于浏览器的所见即所得编辑器，支持目前流行的各种浏览器，由JavaScript写成。其功能配置灵活简单（两行代码就可以将编辑器嵌入网页中），支持AJAX。另一特点是加载速度非常快，如果你的服务器采用的脚本语言是PHP，那还可以进一步优化。




    Gedit


    Gedit是一个GNOME桌面环境下兼容UTF-8的文本编辑器。它使用GTK+编写而成，因此它十分的简单易用，有良好的语法高亮，对中文支持很好，支持包括gb2312、gbk在内的多种字符编码。Gedit在Linux下的一个纯文本编辑器，但你也可以把它用来当成是一个集成开发环境 (IDE)，它会根据不同的语言高亮显现关键字和标识符。




    Scintilla


    Scintilla是一个免费强大稳定的源代码编辑控件，它完全开放源代码。我们能在普通文本编辑器中看到的功能，Scintilla都完全能够实现，而且它还能够提供很多编辑和调试源代码时有用的特殊功能。




    Quanta


    Quanta是一款Web集成开发环境，可以用来开发HTML、XHTML、CSS、XML、PHP以及任何XML风格的语言或脚本语言。它既提供了可视化的所见即所得(WYSIWYG)编辑器，又能允许你手工编辑。它的特点有标记自动补全、标记属性的对话框编辑器、脚本语言变量的自动提示、项目管理、动态预览、PHP调试器、CVS支持、SVN支持等。




    Kompozer


    KompoZer是一个使用方便的支持所见即所得功能的开源HTML编辑器。它基于Nvu(Nvu是一款以Mozilla为核心的开源网页编辑器)，在Nvu的基础上做了些改进。




    cssed


    cssed是一个非常好的CSS编辑器，一个CSS编辑器该有的功能它都有了：CSS编写向导、完备的CSS属性速查、代码高亮、错误排查、代码自动提示等等。即便是一个个没什么经验的CSS新手也能在CSSED的帮助下准确完成自己的编写任务。




    cream


    Cream是一个开源的文本编辑器，它基于Vim，增加了直观的用户界面，包括下拉菜单、工具栏、键盘快捷键和一个基于tab页的多文档接口等强大功能。除此之外，Cream还提供很多插件，帮助你完成所需要的特殊任务。




    Amaya


    Amaya是一套由W3C及INRIA所制作的开源网页浏览器及网页制作软体。它是Grif这套所见即所得的SGML编辑器及Symposia HTML编辑器的后代。Amaya最初的设计成一套HTML及CSS编辑器，但它后来扩展成支持XML相容规格的编辑器，如XHTML, MathML及SVG。
