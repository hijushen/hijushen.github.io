表达式和子查询
全查询：全查询是指返回一个行（只包含一个值）
转换数据类型     cast   数据值-->  字符串
截断一个很长的字符串
EMP_RESUME表中显示简历ASCII格式的前270个字符
SELECT EMPNO,CAST(RESUME AS VARCHAR(270)) 
     FROM EMP_RESUME
     WHERE RESUME_FORMAT = 'ascii'
会发生警告，提示字段被截断，是很正常的选项。
作用：可将空值转换为查询中可处理的其他数据类型。

条件表达式     
     SQL语句中使用CASE表达式以便于处理表的数据表示
     要从ORG表中DEPTNAME列将部门号更改为有意义的字符串
     SELECT DEPNAME,
          CASE DEPTNUMB
               WHEN 10 THEN 'Marketing'
               WHEN 15 THEN 'RESEARCH'
               ELSE
                    'Sales'
               END AS FUNCTION
               FROM ORG
