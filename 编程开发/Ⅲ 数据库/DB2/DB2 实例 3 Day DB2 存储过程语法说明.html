1 DB2 的“开发”中心，可以用JAVA、SQL来创建存储过程，具有很高的移植性；
SQL源代码是CREATE PROCEDURE 语句的一部分，包含代码的那一部分，称为存储过程主体；
2 procedure-name 名在同一数据库同一个模式下，类似于重载方石，不可以相同，即不可以同时同名，同参数个数。不同的参数类型都不可以；

3 SPECIFIC specific-name 唯一指定的名称（名称），可以用存储过程名替代，用来给存储过程 添加备注，可以不添加，但数据库会自动添加一个YYYYXXXRRDDD的时间戳来恒定；

4 DYNAMIC RESULT SETS integer：用于指定存储过程返回结果的最大数量，存储过程没有return 语句，却可以返回结果集；

5 CONTAINS SQL,READS SQL DATA, MODIFIES SQL DATA:用于指定存储过程中SQL访问级别：
     CONTAINS SQL DATA:不可以读取SQL数据，也不可以修改SQL数据；
     READS SQL DATA;只可以读SQL,但不可以修改；
     MODIFIES SQL DATA:不仅可以读，而且可以该，删SQL数据；

6 DETERMINISTIC or NOT DETERMINISTIC 存储过程的动态性：是？ 非？

7 CALLED ON NULL INPUT 不管任何的输入参数是否NULL,任何的OUT或者INOUT参数可以返回一个NULL 或者非空值。检验参数是否为NULL是在过程中进行的；

8 INHERIT SPECIAL REGISTERS:继承专用寄存器；

9 OLD SAVEPOINT LEVEL or NEW SAVEPOINT LEVEL :建立存储点，OLD SAVEPOINT LEVEL  是默认的存储点；

10 LANGUAGE SQL 指定程序的主体使用的是SQL语言。
 
11 EXTERNAL_ACTION  or NO EXTERNA_ACTION : 表示存储过程是否执行一些改变数据库物理状态的活动而不通知数据库管理器，默认EXTERNAL_ACTION, 如果该为NO EXTERNAL_ACTION则表示为最佳优化方案。

12 PARAMETER CCSID:指定所有输出字符串的编码：默认为UNICODE,编码数据库为PARAMETER CCSID UNICODE,其他的数据库默认的为PARAMETER CCSID 3 ASCII;

13 SQL-procedure-body :存储过程的主体；
