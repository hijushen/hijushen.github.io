数据字典命名规范：USER_  ALL_   DBA_   V$前缀：视图是动态执行的视图，包含数据库服务器的性能、存储器和锁的信息。
查看本用户所拥有表的名称：SELECT table_name FROM user_tables；
查看本用户所拥有的不同的对象类型：SELECT DISTINCT object_type FROM user_objects;
查看本用户所拥有的表、视图、同义词和序列：SELECT * FROM user_catalog


Insert语句：一次只能插入一行：INSERT INTO table[(column [, column...])]  VALUES
                                                                            (value [, value...])
字符和日期要用单引号括起来，空值的可以省略，或者使用NULL
特殊值：SYSDATE    &用于变量值的占位符


UPDATE语句:如果需要可以更新多行，修改已经存在的行。UPDATE table SET column = value
