### 增值税(不含税合计, 税额, 价税合计)固定三行的处理.
- 由数据库返回对应行.
- 返回数据增加一个字段 `RowTag`标记, 固定行值分别为1,2,3 . 
- 明细gridPanel的明细 - Model定义中对应的`RowTag` Int类型字段, 若后台没返回, 新增的默认为0.

  - 对combobox类型的处理
    1. 绑定之后数据不显示的原因1: 需要view刷新? `App.GridPanel.view.refresh();`未证实, 
    2. 根据绑定值选取, 防止提交显示的text的值而不是value值. 代码如下:



```javascript
var initComboxByValue = function (comb) {
                        //                        var value = comb.getRawValue();
                        var value = comb.getValue();
                        //                        comb.setValue(value.toString());
 
                        var record = comb.findRecordByValue(value);
                        if (record) {
                            comb.setSelectedItems(record);
                        } else {
                            comb.setValue('');
                        }
 
                    }
var combs = App.ValueAddedTaxPanel.queryBy(function (item) {
                        return item.getXType() == 'combobox';
                    })
 
                    Ext.Array.each(combs, function (item) {
                        initComboxByValue(item);
                    });
```
