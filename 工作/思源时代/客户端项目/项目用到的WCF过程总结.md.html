### 继续NTService
重写的OnStart方法如下:, 因为在交互模式中直接进入HostMange.OpenHost()加载服务过程. 
```
            protected override void OnStart(string[] args)
        {
            int retried = 0;
            while (true)
            {
                if (Environment.UserInteractive)
                {
                    HostManager.OpenHost();
                    break;
                }
                else
                {
                    Process proc = new Process();
                    try
                    {
                        proc.StartInfo.FileName = "copyWCF.HostManage.exe";
                        proc.StartInfo.CreateNoWindow = true;
                        proc.StartInfo.UseShellExecute = false;
                        proc.EnableRaisingEvents = true;
                        proc.Start();
 
                        Thread.Sleep(1000);
                        break;
                    }
                    catch (Exception ex)
                    {
                        retried++;
                        if (retried == 3)
                        {
                            throw ex;
                        }
                        Thread.Sleep(1000);
                    }
                }
            }
            WindowsEventLog.WriteInformationLog("服务启动结束: " + Environment.UserName);
```
