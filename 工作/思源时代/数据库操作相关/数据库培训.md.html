### 系统数据库
- Mater 数据库, 记录了SQL Server 系统所有的系统信息, 这些系统
- Model 数据库, 所有用户数据库和Temp数据创建的末班. 可以将一下信息加入
     
- Msdb 数据. SQL Server Agent需要它来执行安排工作和报警, 记录操作中等操作. **存储计划信息记忆备份和恢复相关的信息**.
- Tempdb 数据库. 用用作系统的临时存储控件, 主要作用:
    - 存储用户建立的临时表和临时存储过程. # 开头表示 ##开头表示
    - 存储用户利用游标说明所筛选出来的数据
    - 为数据排序创建临时表
    - 存储用户说明的全局变量值.
    - 说明: 退出SQL Server时, 用户Tempdb建立的对象删除.
