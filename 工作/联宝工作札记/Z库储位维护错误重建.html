实例: 因为材料大小维护错误, 与Z 储位维护时错误. 因为当前Z 储位解除bug 问题,需要手动删除link重建.
--解决方案: 1, 先出库, 解决库存. 2, 维护料号大小料盘信息. 3,解除z_link 信息.即可.

SELECT ROWID,A.* FROM CIMR.R_MMS_STOCK_SMT_ZSTORE_T A; --Z库库存信息
SELECT ROWID,B.* FROM CIMR.R_MMS_STOCK_SMT_ZDETAIL_T B
WHERE B.REEL_ID = '12345H45200CAK';
SELECT D.*,ROWID FROM CIMC.C_MMS_STOCK_Z_LOCATION_T D;
SELECT E.*,ROWID FROM CIMC.C_MMS_STOCK_Z_LINK_T E;
WHERE E.PART_NO = 'CY000002M00'
